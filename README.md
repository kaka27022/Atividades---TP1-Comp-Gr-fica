# Exercícios de Computação Gráfica

Repositório com implementações práticas dos principais conceitos da disciplina **Computação Gráfica**, utilizando **OpenGL/GLUT**.  
Cada exercício aborda um tema específico, como transformações geométricas, modelos de cor, interação por teclado e renderização básica.

---

## Objetivo

Explorar e compreender os fundamentos da Computação Gráfica através de implementações simples e interativas.  
Os programas foram desenvolvidos com foco didático, servindo como base para quem está aprendendo OpenGL.
