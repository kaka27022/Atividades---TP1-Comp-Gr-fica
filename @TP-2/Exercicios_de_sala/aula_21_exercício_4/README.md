# Atividade: Back-Face Culling 

Este projeto implementa o desafio prático remoção de superfícies escondidas.

## Como Compilar e Executar
No terminal (dentro da pasta do projeto):

1. **Para compilar:**
    ```bash
   make

2. **Para executar:**
    ```bash
   make run

| Tecla       | Ação                     | Descrição                                  |
| ----------- | ------------------------ | ------------------------------------------ |
| **C**       | Ativar/Desativar Culling | Liga ou desliga o `glEnable(GL_CULL_FACE)` |
| **F**       | Alternar Front Face      | Alterna entre `GL_CCW` (padrão) e `GL_CW`  |
| **↑ ↓ ← →** | Rotação                  | Rotaciona o cubo nos eixos **X** e **Y**   |
| **ESC**     | Sair                     | Encerra a aplicação                        |
